## Annonce

Nous sommes ravis de vous inviter au [BitDevs Meetup](/fr/about) de ce mois à Cotonou !

Le mois dernier, nous avons connecté notre instance BTCPay Server à une application web et réalisé ensemble des paiements Bitcoin en direct pour mieux comprendre l’intégration des paiements dans les projets.

Ce mois-ci, nous changeons de cap pour explorer les outils de collaboration open source.

- À travers un atelier pratique sur Git & GitHub, nous allons découvrir comment :

- Créer et configurer un compte GitHub

- Installer et utiliser Git comme outil de versioning

- Comprendre la différence entre forker et cloner un dépôt

- Explorer les notions d’issues et de pull requests

- Contribuer à un projet open source avec un processus inspiré de Bitcoin Core

Ce sera l’occasion parfaite pour :

- Acquérir des compétences pratiques sur Git et GitHub

- Découvrir comment contribuer au développement open source Bitcoin

- Échanger avec d’autres développeurs et passionnés de la communauté

Ne manquez pas cette opportunité !

Réservez dès maintenant votre place et rejoignez la communauté BitDevs locale pour une session riche en apprentissage et en pratique.

Nous remercions chaleureusement notre sponsor [Btrust](https://btrust.tech) pour son soutien à l’émergence de la Communauté de Développeurs Bitcoin au Bénin.
