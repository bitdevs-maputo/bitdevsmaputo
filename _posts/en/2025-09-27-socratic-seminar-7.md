## Announcements

We are excited to invite you to this month’s [BitDevs Meetup](/en/about) in Cotonou!

Last month, we connected our BTCPay Server instance to a web application and ran live Bitcoin payments to better understand how to integrate Bitcoin into real projects.

This month, we are shifting gears to explore open-source collaboration tools.

Through a hands-on workshop on Git & GitHub, we will learn how to:

- Create and set up a GitHub account

- Install and use Git as a version control tool

- Understand the difference between forking and cloning a repository

- Explore issues and pull requests

- Contribute to an open-source project following a process inspired by Bitcoin Core

This will be the perfect opportunity to:

- Gain practical skills with Git and GitHub

- Discover how to contribute to Bitcoin open-source development

- Connect and exchange with other developers and enthusiasts

Don’t miss out!

Reserve your spot today and join the local BitDevs community for a session full of learning and practice.

We warmly thank our sponsor [Btrust](https://btrust.tech) for supporting the growth of Bitcoin Developers Community in Benin.
